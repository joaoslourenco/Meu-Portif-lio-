Sobre Mim

Tenho 16 anos, sou de Belo Horizonte e atualmente curso o segundo ano do técnico em Tecnologia da Informação no Colégio Cotemig. Minha paixão pela tecnologia começou cedo e hoje já trilho meu caminho no desenvolvimento de soluções digitais, sempre com foco em unir design moderno, usabilidade e um código limpo.


Tenho experiência com banco de dados, projetos de back-end, criação de sites, integrações com API e também em redes. Acredito que a tecnologia é um dos maiores instrumentos de transformação do presente e do futuro, e por isso me dedico a estudar constantemente novas ferramentas e conceitos que me permitam evoluir como profissional e como pessoa.


Mesmo jovem, sei que estou construindo as bases para uma carreira sólida e promissora. Meu objetivo é continuar crescendo, explorando diferentes áreas da tecnologia e desenvolvendo projetos que realmente possam impactar e gerar valor.

Habilidades

HTML

Linguagem de marcação para estruturar o conteúdo de páginas web. Uso para organizar textos, imagens, links, formulários e elementos da interface.

CSS

Linguagem de estilo para definir a aparência de páginas web. Uso para controlar cores, tipografia, espaçamento, layout e tornar o site responsivo.

JavaScript

Linguagem de programação essencial para a web. Uso para adicionar interatividade, dinamismo e integrações em páginas, além de criar aplicações completas no front-end e back-end.

Python

Linguagem de programação simples e multifuncional. Uso para desenvolvimento web, automação, análise de dados, ciência de dados e projetos de inteligência artificial.

C#

Linguagem de programação da Microsoft para aplicações modernas. Uso para desenvolver sistemas corporativos, aplicações desktop e jogos com Unity.

SQL

Linguagem de consulta para gerenciar bancos de dados relacionais. Uso para criar e estruturar tabelas, inserir e consultar registros e manipular dados de forma segura.
